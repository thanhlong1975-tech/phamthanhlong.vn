# Cadivi Resources

This folder is the official source repository for Cadivi documents.

## Rules

- Every file stored here must be an official source.
- Never edit original documents.
- Processed data must be stored outside resources.
- Every Product Family and Product must reference documents from this folder.

## Storage areas

- catalogue/: official catalogues
- price-lists/: official price lists
- datasheets/: official datasheets
- standards/: official standards and related documents
- images/: official images
